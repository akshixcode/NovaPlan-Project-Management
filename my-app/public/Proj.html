Project Name

Responsive Portfolio Website with Admin Dashboard

A fully responsive portfolio website with an integrated content management system that allows the owner to update portfolio items, blog posts, and manage contact messages without touching code.

Folder Structure

Root Directory

/portfolio-website

Contains all project files and subdirectories

Main Folders

  • /public - Static assets (images, icons, fonts)
  • /src - Source code
  • /admin - Admin panel files
  • /database - Database schemas and scripts

Source Code Structure

  • /src/components - React components
  • /src/pages - Page components
  • /src/styles - CSS and SCSS files
  • /src/utils - Helper functions

Tech Stack Used

Frontend Technologies

HTML5 CSS3 JavaScript (ES6+) React.js Bootstrap 5 SASS/SCSS

Backend & Database

Node.js Express.js MongoDB Mongoose ODM JWT Authentication
